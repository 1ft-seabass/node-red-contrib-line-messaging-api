LINE Messagin APIのNode-REDのノードです。

## 概要

LINE Messagin APIを利用できるNode-REDのノードです。

以下のAPIを利用できます。

* LINE Messaging API
    * Reply Message
    * Push Message
* LINE Notify

## インストール

```
npm i node-red-contrib-line-messaging-api
```
